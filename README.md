# Dress-up
Dress up spēlīte kur var apģērbt tēlu vairākos veidos, mainot drēbes, tās veidu, slēpjot drēbju veidu, utt. Var arī mainīt tēla platumu un garumu.
- [ ] Vairāki drēbju veidi
- [ ] Vismaz 3 drēbes katrā
- [ ] 2 tēli
- [ ] Var mainīt garumu, platumu
- [ ] Var iziet no aplikācijas
- [ ] Var ieiet scene kur ir galvenā spēle
- [ ] Var doties atpakaļ uz main menu

# Dress-up
Dress up spēlīte kur var apģērbt tēlu vairākos veidos, mainot drēbes, tās veidu, slēpjot drēbju veidu, utt. Var arī mainīt tēla platumu un garumu.
Drēbes ir samētātas apkārt, ja ileiek skapī tās visas būs vienā vietā.
Ja grib mainīt drēbes, tās ir jāieliek skapī.
Nevar būt vairāk par 1 apģērbu katrā sadaļā.

- [x] Vismaz 3 drēbes katrā
- [x] 2 tēli
- [x] Var mainīt garumu, platumu
